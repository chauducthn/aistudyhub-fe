# Asset folders

Use these folders for Figma exports so UI work can match the design more closely.

- `icons/`: small SVG/PNG icons, preferably SVG.
- `logos/`: brand marks and logo variants.
- `illustrations/`: large decorative or conceptual artwork used in auth/hero sections.
- `images/`: screenshots, product previews, thumbnails, photos, and dashboard mockups.

Naming convention:

- Use lowercase kebab-case: `login-hero-illustration.png`, `google-icon.svg`.
- Export raster images at 2x when possible.
- Prefer transparent PNG/WebP for illustrations and SVG for icons/logos.
- Keep full-page Figma screenshots only when they are intentionally used as reference assets.
